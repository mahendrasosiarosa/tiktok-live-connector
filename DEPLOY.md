# Deploy TikTok LIVE Connector ke Railway (GRATIS)

## Langkah 1 — Buat akun GitHub
1. Buka https://github.com
2. Klik "Sign up" → buat akun gratis
3. Verifikasi email

## Langkah 2 — Upload project ke GitHub
1. Buka https://github.com/new
2. Nama repo: `tiktok-live-connector`
3. Set ke **Public** → klik "Create repository"
4. Di halaman repo, klik "uploading an existing file"
5. Upload SEMUA file dari folder ini (drag & drop)
6. Klik "Commit changes"

## Langkah 3 — Deploy ke Railway
1. Buka https://railway.app
2. Klik "Login with GitHub"
3. Klik "New Project" → "Deploy from GitHub repo"
4. Pilih repo `tiktok-live-connector`
5. Railway otomatis detect Node.js dan deploy!
6. Tunggu ~2 menit → klik "Generate Domain"
7. Dapat URL seperti: `https://tiktok-live-connector-xxx.railway.app`

## Selesai!
- Kamu & teman buka URL itu di browser HP
- Input username TikTok → Connect
- Semua fitur jalan dari HP tanpa PC!

## Catatan
- Railway gratis: 500 jam/bulan (cukup untuk ~16 jam/hari)
- Kalau mau unlimited, upgrade $5/bulan
